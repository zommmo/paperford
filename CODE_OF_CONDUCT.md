# Code of Conduct

Paperford is a small open source project. Keep discussions practical, respectful, and focused on improving the software.

## Expected Behavior

- Be respectful of other contributors and users.
- Give clear technical feedback.
- Avoid personal attacks, harassment, or discriminatory language.
- Do not post private data, API keys, private EPUB files, or credentials.

## Reporting

If you see behavior that makes participation unsafe or unproductive, contact the maintainer or open a GitHub issue with non-sensitive details.
